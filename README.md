# Tic_Tac_Toe-in-Python
The Tic Tac Toe project in Python is a simple console-based game developed using the Python programming language. This project allows two players to play the classic Tic Tac Toe game by marking X and O symbols on a 3x3 grid. It demonstrates concepts such as loops, functions, lists, conditional statements, and game logic implementation.

## Features
* Two-player game mode
* Interactive game board
* Player turn management
* Win and draw detection
* Input validation
* User-friendly console interface
* Simple and efficient game logic

## Technologies Used
* Python Programming Language
* Functions
* Lists and Loops
* Conditional Statements
* Basic Game Logic

## Project Structure
* `main.py` → Main game execution file
* `game.py` → Game logic and functions
* `board.py` → Board display and updates
* `utils.py` → Helper functions
* `README.md` → Project documentation
